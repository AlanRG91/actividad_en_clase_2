# Adivina el Número

Este es un proyecto sencillo de Python para practicar lógica de programación básica y manejo de entradas de usuario.

## 🚀 Descripción
El programa genera un número aleatorio entre 1 y 10 y pide al usuario que intente adivinarlo, proporcionando pistas en cada intento.

## 🛠️ Funcionamiento
1. El script inicia un bucle `while`.
2. Solicita un número entero al usuario.
3. Evalúa si el número es mayor, menor o igual al objetivo.
4. Cuenta la cantidad de intentos realizados.
5. Finaliza cuando el usuario acierta.

## 💻 Ejecución
Para correr el código, utiliza:
```bash
python adivinanza.py
